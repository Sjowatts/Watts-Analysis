Squash Dissertation: Rally Intensity

Research into time between shots, shots per minute, player profiling, and how rally intensity relates to match outcomes and world ranking in elite squash, using 122 matches from six PSA World Tour events.

Project snapshot

Overview

Dataset 122 matches
Events Six PSA World Tour events
Metrics TB and SPM
Outputs Poster and dissertation

Project focus

The dissertation examines whether rally intensity can be measured effectively, how players can be profiled using those measures, and how the resulting patterns relate to outcome and ranking.

Method and takeaway

Video analysis and Angles software were used to code the dataset. The study found that higher rally intensity aligned with faster time between shots, higher shots per minute, and stronger match performance, particularly among top-ranked players.
