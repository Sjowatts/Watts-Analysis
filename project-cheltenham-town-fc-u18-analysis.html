Project focus

This work was built around giving academy coaches more granular insight into player performance, team dynamics, and set-piece detail, while keeping the output structured enough to be useful immediately.

What the work supported

Interactive reports highlighted important metrics and trends to support decisions on player rotations, match strategy, and defensive improvements. The original public archive did not include supporting visuals, so this page now carries the case-study context cleanly until publishable assets are available.

A key outcome from this work was identifying and addressing set-piece vulnerabilities, which contributed to stronger defensive resilience.
